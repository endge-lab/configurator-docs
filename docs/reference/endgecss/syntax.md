# Синтаксис и значения

Синтаксис EndgeCSS близок к CSS/SCSS: поддерживаются обычные rules, вложенность, `&`, комментарии, custom properties и `var()`.

```css
.flight-card {
  --accent: #2563eb;
  color: var(--accent);

  &:state(delayed) > Text {
    font-weight: 600;
  }
}
```

Значения разбираются на нейтральные слова, строки, разделители и функции. Язык сохраняет знакомую форму CSS, но не наследует весь SCSS.

Не поддерживаются:

- SCSS-переменные `$variable`;
- mixins;
- imports;
- пользовательские функции;
- property nesting.

Ограниченный синтаксис нужен для одинаковой интерпретации разными renderer.
