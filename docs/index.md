# Что такое Endge

Endge — платформа для описания приложений через доменные документы. В домене хранятся структура данных, запросы, преобразования, вычисления, компоненты и связи между ними. Конфигуратор предоставляет интерфейс для создания и редактирования этих документов, а ядро компилирует их и исполняет в runtime.

## Из чего состоит система

- **Конфигуратор** — рабочая среда, в которой пользователь управляет проектом и его документами.
- **Домен** — сохраняемое описание приложения: типы, запросы, представления данных, компоненты и другие сущности.
- **Компилятор** — проверяет source-документы и преобразует их в исполняемые программы.
- **Runtime** — создаёт живые экземпляры программ, связывает их с данными и управляет их жизненным циклом.
- **Адаптер представления** — отображает результат в конкретной среде, например во Vue-приложении.

## Главный принцип

В Endge сохраняется не готовый UI и не произвольное состояние runtime, а исходное описание — **source**. Из него можно получить проверенную программу и исполнить её в подходящем окружении.

```text
source → compiler → program → runtime → presentation
```

Такое разделение позволяет независимо развивать редактор, язык документов, исполнение и способы отображения.
