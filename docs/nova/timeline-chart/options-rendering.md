# rendering

Rendering options и task profiles отвечают за стоимость отрисовки. Основные рычаги: высота задач, LOD для текста, compiled recipes и отключение лишней графики.

## Практика

Для тысяч задач избегайте тяжелых custom templates на каждую задачу. Сначала попробуйте compiled recipe или простой schema profile.
