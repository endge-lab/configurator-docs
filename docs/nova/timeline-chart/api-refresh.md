# refresh

`refresh` запрашивает мягкое обновление runtime-состояния. Обычно он не нужен после `data`, `options`, `add`, `update` или `remove`.

## Когда применять

Используйте `refresh` после внешних изменений, которые не проходят через публичные data/options API, например после обновления ресурсов templates.
