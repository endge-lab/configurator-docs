# flush behavior

Обычные команды API применяются сразу в текущем runtime cycle. `batch` объединяет несколько команд. `executor` может отложить применение, чтобы снизить стоимость частых изменений.

## Выбор режима

- Разовая команда UI: прямой `update`.
- Несколько связанных команд: `batch`.
- Live-поток: `batch` вместе с `executor`.
