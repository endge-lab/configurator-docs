# Горячие клавиши редактора

Ниже перечислены горячие клавиши, которые добавляет редактор Endge. Стандартные комбинации браузера и Monaco не дублируются, если конфигуратор не меняет их поведение.

| macOS | Windows / Linux | Действие | Контекст |
| --- | --- | --- | --- |
| `Command+S` | `Control+S` | Сохранить активный документ | Активная вкладка документа |
| `Command+W` | `Control+W` | Закрыть активную вкладку | Рабочая область конфигуратора |
| `Command+N` | `Control+N` | Открыть создание документа | Вне текстовых полей |
| `Command+Enter` | `Control+Enter` | Запустить текущий документ и активировать Runtime Tree | Редактор Project, Composition, Component SFC или Store |
| `Escape` | `Escape` | Активировать Project и выйти из области Runtime Preview | Активен Runtime Tree |
| `Command+B` | `Control+B` | Открыть определение Endge, на которое указывает ссылка | Source-редактор Endge, курсор установлен на поддерживаемой ссылке |

## Приоритет Escape

Диалоговые окна и контекстные меню имеют приоритет над навигацией между рабочими областями. Если такое окно открыто, первое нажатие `Escape` закрывает его. Нажмите `Escape` ещё раз, чтобы вернуться из Runtime Tree к Project.

## Источник истины

Глобальные горячие клавиши регистрируются в `src/features/endge-ide/model/core/endge-ide-hotkeys.ts`. Комбинация для перехода к определению в source-редакторе Endge регистрируется в `src/features/endge-ide/tools/source-editor/use-endge-source-monaco.ts`.
