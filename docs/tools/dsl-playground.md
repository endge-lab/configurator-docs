# DSL Playground

**DSL Source Playground** - редактор исторического JSX source.

## Назначение

Позволяет просматривать, редактировать и форматировать старый JSX source без сохранения в домен.

## Как пользоваться

- Открыть плагин через меню (например, «DSL Песочница»).
- Доступны готовые source examples из выпадающего списка.
- Содержимое редактора сохраняется в `localStorage` и восстанавливается при следующем открытии.
- Кнопка форматирования приводит JSX к единому стилю.

Compile and runtime preview are intentionally unavailable. Legacy DSL documents are data-only.
