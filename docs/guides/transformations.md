# Преобразование данных

Для преобразований используются DataView и Converter. Они решают разные задачи.

## DataView

DataView преобразует набор данных: выбирает источник, проходит по строкам, объединяет коллекции и формирует новую структуру.

```ts
defineDataView({
  mode: 'pipeline',
  steps: [
    from('raw').as('row'),
    map({
      id: path('row.id'),
      title: path('row.name'),
    }),
  ],
})
```

## Converter

Converter преобразует отдельное значение или формат. Например, приводит дату к строке отображения или код статуса к подписи.

## Как выбрать

| Задача | Механизм |
| --- | --- |
| Изменить структуру коллекции | DataView |
| Объединить две коллекции | DataView |
| Вычислить поле для каждой строки | DataView |
| Преобразовать одно значение | Converter |
| Построить зависимый граф вычислений | Computation |

Подробности: [DataView](/reference/data-view) и [Converter](/reference/converter).
